		Installing Jenkins
1. Install Java
	#yum -y install java-11-openjdk
	#java -version
	--> remove old version jdk (#yum -y remove package name) OR
	--> Configure the alternative java version
		#alternatives --config java 	--> Select your appropriate java 
		#java -version
	
2. Set JAVA_HOME variable path
	#cd /usr/lib/jvm/java-11<tab>
	#pwd --> copy the path
	#vi /root/.bash_profile
		JAVA_HOME=(paste the copied path)
		PATH=$PATH:$HOME/bin:$JAVA_HOME
	#. /root/.bash_profile OR 
	#source /root/.bash_profile (to refresh)

3. Installing Jenkins
	#cd /etc/yum.repos.d
	--> brows the official web site of jenkins for genuene jenkins repos
	#wget <past url for repo>
	#rpm --import <past url for key>
	#yum -y install jenkins
	#rpm -q jenkins

4. start the jenkins service
	#systemctl start jenkins
	#systemctl enable jenkins
	#systemctl status jenkins
	
5. login to the jenkins server (i.e. accessing jenkins dashboard)
	--> default port of jenkins = 8080
	--> ifconfig -->(note down the ip)
	#firefox (any browser)
	--> http://192.168.100.152:8080
	--> user given location to get password
